How to Use

URL Pattern

Generate placeholder images by using a simple URL pattern:

https://fpoimg.com/{width}x{height}?text={text}bg_color={bg_color}&text_color={text_color}

Basic Usage
https://fpoimg.com/800x600

Creates an 800×600 pixel image with default colors.

With a caption
https://fpoimg.com/800x600?text=Caption

Creates an 800×600 pixel image with a caption

Custom Colors
https://fpoimg.com/400x300?bg_color=4361ee&text_color=ffffff

Creates a 400×300 pixel image with blue background and white text.

Square Image
https://fpoimg.com/500

Creates a 500×500 pixel square image.
