Named Colors

Use any of the 148 CSS named colors instead of hex codes. Just type the color name in your URL — it's that easy.

🎨 How to Use Named Colors

Named colors work everywhere hex colors work — in the URL path or as query parameters:

/400x300/tomato/white
/300/steelblue
/800x600?bg_color=coral&text_color=navy
/500x300?gradient=tomato,steelblue
